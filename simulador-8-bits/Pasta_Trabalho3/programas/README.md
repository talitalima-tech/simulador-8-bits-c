# Simulador de Computador de 8 bits em C

Projeto desenvolvido na disciplina de **Arquitetura de Computadores**, com o objetivo de simular o funcionamento de um computador de **8 bits**, incluindo CPU, ULA e Unidade de Controle.

O simulador processa **instruções de máquina** por meio da implementação do **ciclo de instrução**, utilizando micro-operações semelhantes às de arquiteturas reais.

---

## 🎯 Objetivo do Projeto

- Completar a implementação de um computador de 8 bits em linguagem **C**
- Simular o processamento de instruções de máquina
- Aplicar conceitos fundamentais de:
  - Arquitetura de Computadores
  - Organização de CPU
  - Unidade Lógica e Aritmética (ULA)
  - Unidade de Controle (UC)

O desenvolvimento foi realizado a partir de um **esqueleto de código fornecido pelo professor**, conforme especificação da disciplina.

---

## ⚙️ Funcionalidades Implementadas

### 🔹 Unidade Lógica e Aritmética (ULA)
- Implementação das operações aritméticas e lógicas
- Baseada no trabalho anterior de **Simulador de ULA**

### 🔹 Ciclo de Instrução da CPU
Foram implementados os principais ciclos de funcionamento da CPU:

- **Ciclo de Busca (Fetch)**
- **Ciclo Indireto**
- **Ciclo de Execução**
- **Ciclo de Interrupção**

Cada ciclo é composto por **micro-operações** controladas pela **Unidade de Controle (UC)**.
